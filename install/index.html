Run It Yourself

TRMCompare is a static site. No server-side code, no database, no containers. Just files and a browser.

Quickest option: just bookmark it

The GitHub Pages version works everywhere with internet access. Bookmark it and you're done.

Bookmark TRMCompare
Recommended

Python (one command)

Every Mac and most Linux systems have Python 3 installed. Clone and serve:

git clone https://github.com/lbruton/TRMCompare.git
cd TRMCompare
python3 -m http.server 8080

Open http://localhost:8080 in your browser. That's it.

To stop the server, press Ctrl+C in the terminal.

Alternative

Download ZIP

No git? No problem.

  1. Go to the ZIP download
  2. Extract the ZIP
  3. Open a terminal in the extracted folder
  4. Run python3 -m http.server 8080
  5. Open http://localhost:8080
Alternative

Node.js

If you have Node.js installed:

npx serve .

Or use any static file server: nginx, Apache, Caddy, IIS — just point it at the folder.

Windows

Windows without Python

Don't have Python on Windows? Use the built-in PowerShell approach:

cd TRMCompare
Start-Process "http://localhost:8080"
python -m http.server 8080

Or install Python from python.org — check "Add to PATH" during install.

Why do I need a server? TRMCompare uses ES modules (import/export), which browsers block when opening files directly (file://) due to CORS security. Any HTTP server works — the server just delivers the files, all processing happens in your browser.
